개인정보처리방침

WebGuard QA Chrome Extension

요약: WebGuard QA는 어떠한 개인정보도 수집하지 않으며, 모든 데이터는 사용자의 브라우저에만 저장됩니다.

1. 수집하는 정보

WebGuard QA는 어떠한 개인정보도 수집하지 않습니다.

확장 프로그램이 생성하는 모든 데이터는 다음과 같습니다:

  • 스캔한 웹사이트의 URL
  • 보안 스캔 결과 (취약점 정보)
  • 스캔 실행 날짜 및 시간
  • 사용자가 설정한 스캔 옵션

이러한 데이터는 오직 사용자의 로컬 브라우저(Chrome Storage)에만 저장되며, 외부로 전송되지 않습니다.

2. 데이터 저장 및 사용

모든 스캔 결과는 Chrome의 로컬 스토리지(chrome.storage.local)에 저장됩니다:

  • 저장 위치: 사용자의 컴퓨터 (브라우저 내부)
  • 저장 목적: 스캔 히스토리 제공, 대시보드 표시
  • 저장 기간: 사용자가 삭제하거나 확장 프로그램을 제거할 때까지
  • 외부 전송: 절대 없음

3. 권한 사용 설명

WebGuard QA는 다음 권한을 사용합니다:

  • activeTab: 현재 탭에서 보안 스캔을 수행하기 위해 사용
  • storage: 스캔 결과를 사용자의 브라우저에 로컬 저장
  • tabs: 대시보드 및 도움말 페이지를 새 탭으로 열기 위해 사용
  • scripting: 웹페이지의 DOM 요소를 분석하기 위해 사용
  • contextMenus: 우클릭 메뉴에 "스캔" 옵션 추가
  • notifications: 심각한 보안 문제 발견 시 알림 표시
  • downloads: 스캔 결과를 JSON 파일로 내보내기

이러한 권한은 오직 확장 프로그램의 기능을 수행하기 위해서만 사용되며, 개인정보 수집이나 추적 목적으로 사용되지 않습니다.

4. 제3자 공유

WebGuard QA는 사용자 데이터를 제3자와 절대 공유하지 않습니다.

  • 외부 서버로 데이터 전송 없음
  • 분석 도구 사용 없음
  • 광고 네트워크 연결 없음
  • 추적 쿠키 사용 없음

5. 데이터 보안

모든 데이터는 Chrome의 안전한 스토리지 API를 통해 저장됩니다:

  • 데이터는 사용자의 Chrome 프로필에 암호화되어 저장
  • 다른 확장 프로그램이나 웹사이트는 접근 불가
  • 사용자만이 자신의 데이터를 확인하고 삭제 가능

6. 사용자 권리

사용자는 언제든지:

  • 대시보드 설정에서 모든 스캔 히스토리 삭제 가능
  • 확장 프로그램 제거 시 모든 데이터 자동 삭제
  • Chrome 설정에서 개별 데이터 관리 가능

7. 아동 개인정보

이 확장 프로그램은 웹 개발자를 위한 도구이며, 의도적으로 아동 대상 서비스가 아닙니다. 13세 미만 아동의 개인정보를 수집하지 않습니다.

8. 개인정보처리방침 변경

이 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경 사항이 있을 경우 확장 프로그램 업데이트 노트를 통해 공지합니다.

9. 오픈소스
